Typography

This section describes the fonts that we use in our dashboard, different typography elements, their sizing and styling.

Font Family

Syncopate
Montserrat
Noto

In our dashboard we use 3 Fonts - Syncopate, Monstserrat and Noto. The first one, being more stylish is used in headings and titles. For content and for readability, we use Montserrat which is very similar to title type. For data representation and charts, we use Noto, a serif font that pairs well with Montserrat.

Type Scale

Below are presented type scales for all the headings and paragraphs that we use in the dashboard. All headings are used in order to highlight their sections, going from the highest importance to the lowest.

Heading One

48px, Syncopate, Bold, Uppercase.

Used for banner heading

Heading Two

28px, Syncopate, Bold, Uppercase.

Used for page headings

Heading Three

16px, Syncopate, Normal, Uppercase.

Used for page headings

List Items

14.4px, Syncopate, Normal, Lowercase.

Used for side list titles

Paragraph

15px, Montserrat, Normal.

Used for content text

Chart Description

16px, Noto, Normal.

Used for chart content

page layout

This section descripes the layouts for the dashboard.

Desktop Layout

The dashboard will have a top bar with main navigation section, logo and company banner. On the left side we wilil have the dahboard navigation bar. The right side will contain the dashboard's content with data charts.

Mobile Layout

In mobile view the page will have a top bar containing logo and a hamburger icon. The drop down menu will opeon upon clicking on hamburger sign.

Grid Break Points

The dashboard will have 2 main break points for responsive display. 400px for mobile devices and 800px for tablets.

Data Visualization

This section illustrates different type of charts and graphs we use in our dashboard.

Bar Charts

We will utilize two types of Bar Charts in the dashboard. Normal and group bar Horizontal. The font used will be Noto as described in the font section of the library.

Normal Bar Chart

This bar chart displays different metrics that compare values in repsect of eachother or over time.

Horizontal Group Bar Chart

This Bar chart is a multiple value cahrt that is similar to normal bar chart, but in horizontal format going left to right, illustrating comparison for multiple data values.

Line Graphs

For the dashboard, we will be using Normal Line Graphs for data comparison.

Scatter Plot

The dashboard will utilize Scatter plot to illustrate general trends or relationship between variables.

Single Value Charts

The dashboard will utilize single value charts to provide a quick glance reference to a key value.
